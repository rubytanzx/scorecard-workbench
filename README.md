# IDA Scorecard Workbench

An interactive data exploration tool for the WBG Corporate Scorecard FY24–FY30, built with React 19 + Vite.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

## Tech Stack

- **React 19** + **TypeScript**
- **Vite 6**
- **Tailwind CSS v4**
- **D3** (globe + map visualisations)

## Project Structure

```
app/          # Page components
components/   # UI components
lib/          # Data and utilities
public/       # Static assets (SVGs, GeoJSON, images)
context/      # Scorecard context docs for AI prompting
```

## Build

```bash
npm run build
npm run preview
```
